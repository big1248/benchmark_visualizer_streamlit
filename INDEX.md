# 🎯 LLM 벤치마크 시각화 도구

> 다양한 LLM 모델의 벤치마크 결과를 테스트명, 모델, 법령/비법령, 과목별로 비교 분석하는 종합 시각화 도구입니다.

---

## 🚀 3단계로 시작하기

```bash
# 1. 패키지 설치
pip install -r requirements.txt

# 2. 데이터 확인 (선택)
python validate_data.py /mnt/project

# 3. 실행!
streamlit run llm_benchmark_visualizer.py
# 또는
./run.sh
```

**그게 전부입니다!** 브라우저가 자동으로 열립니다. 🎉

---

## 📊 주요 기능

### 1. 📈 다차원 분석
- **테스트별**: 산업안전기사, 방재기사, 건설안전기사, 방재안전직 등
- **모델별**: Claude-3.5-Sonnet, GPT-4o, GPT-4o-Mini, Claude-3.5-Haiku
- **법령 구분**: 법령 문제 vs 비법령 문제
- **과목별**: 안전관리론, 전기위험방지기술, 건설안전기술 등
- **상세도별**: detailed vs summary 방식

### 2. 🎨 풍부한 시각화
- **히트맵**: 모델×테스트, 모델×과목 성능 매트릭스
- **막대 그래프**: 정확도, 성능 비교
- **산점도**: 정확도 vs 처리속도
- **박스플롯**: 정답률 분포
- **파이 차트**: 법령/비법령 비율
- **라인 차트**: 과목별 성능 추이

### 3. 🔍 강력한 필터링
- 여러 조건 동시 적용
- 실시간 업데이트
- 인터랙티브 차트

### 4. 📤 데이터 내보내기
- 테이블 데이터 CSV 다운로드
- 차트 이미지 저장
- 보고서 작성 지원

## 🎯 사용 시나리오

### 시나리오 1: 최적 모델 선택
**질문**: "어떤 모델을 사용해야 할까?"

**솔루션**:
1. "📊 전체 요약" → 모델별 정확도 순위
2. "🏆 모델별 비교" → 정확도 vs 속도
3. 비용과 성능의 균형점 찾기

**결과 예시** (실제 데이터 기준):
- **최고 정확도**: GPT-4o (75.18%)
- **균형형**: Claude-3.5-Sonnet (67.73%)
- **저비용**: Claude-3.5-Haiku (60.28%), GPT-4o-Mini (62.77%)

---

### 시나리오 2: 법률 AI 개발
**질문**: "법령 문제 처리에 강한 모델은?"

**솔루션**:
1. 사이드바 → "법령 구분" → "법령"
2. "⚖️ 법령/비법령 분석" 확인
3. 모델별 법령 문제 정확도 비교

**활용**: 법률 챗봇, 법무 자동화 시스템

---

### 시나리오 3: 과목별 전문화
**질문**: "특정 과목에 최적화된 모델은?"

**솔루션**:
1. "📖 과목별 분석" 탭
2. 관심 과목 선택
3. 모델별 성능 비교

**활용**: 과목별 맞춤형 교육 AI

---

### 시나리오 4: 비용 최적화
**질문**: "예산 내에서 최고 성능은?"

**솔루션**:
1. "모델별 비교" → 정확도 vs 속도
2. 정확도, 처리 시간, 비용 수준 종합 고려
3. ROI 계산

**활용**: 대량 처리 시스템, 비용 민감 프로젝트

## 💡 빠른 팁

### 효율적인 분석 흐름
```
1. 전체 현황 파악 (전체 요약)
   ↓
2. 모델 비교 (모델별 비교)
   ↓
3. 상세 분석 (테스트별/과목별)
   ↓
4. 특화 분석 (법령/비법령)
```

### 필터 활용 예시
```python
# 예시 1: 특정 모델만 비교
사이드바 → 모델 → GPT-4o, Claude-3.5-Sonnet 선택

# 예시 2: 법령 문제만 분석
사이드바 → 법령 구분 → 법령 선택

# 예시 3: 복합 조건
테스트: 산업안전기사
모델: GPT-4o
법령: 법령
→ 특정 테스트의 법령 문제에서 특정 모델 성능
```

### 그래프 활용
- **확대**: 드래그로 영역 선택
- **축소**: 더블 클릭
- **정보**: 마우스 오버
- **저장**: 📷 버튼 클릭
